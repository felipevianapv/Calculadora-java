# Projetos Java - Estudos

Projetos simples em Java para praticar lógica de programação.

## Projetos

Calculadora.java
- Calculadora com operações matemáticas básicas.

Tabuada.java
- Mostra a tabuada de um número informado pelo usuário.

MediaAluno.java
- Calcula a média de três notas e informa aprovação, recuperação ou reprovação.

ParOuImpar.java
- Verifica se um número é par ou ímpar.

## Tecnologias utilizadas
Java
Scanner
Estruturas condicionais (if / else)
Laços de repetição (for)
