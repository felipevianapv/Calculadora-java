# Projetos Java - Estudos

Projetos simples em Java para praticar lógica de programação.

## Projetos

Calculadora.java
- Calculadora simples com operações matemáticas.

Tabuada.java
- Mostra a tabuada de um número digitado pelo usuário.

MediaAluno.java
- Calcula a média de três notas e informa se o aluno foi aprovado, recuperação ou reprovado.

## Tecnologias
Java
Scanner
Estruturas condicionais (if / else)
Laços de repetição (for)
